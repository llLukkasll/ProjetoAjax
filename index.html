Óla meu nome e Ajax sou professor particular de programação tenho mais de 15 anos de profissao, já atuei em varias areas da tecnologia, tenho muita experiencia para ser compartilhada. programo em 4 linguagens sao elas Python, Java, C# e R vou falar um pouco de cada:

Python é uma linguagem de programação de alto nível e de propósito geral. Sua filosofia de design enfatiza a legibilidade do código com o uso de recuo significativo.

Java é uma linguagem de programação de alto nível, baseada em classes e orientada a objetos, projetada para ter o menor número possível de dependências de implementação.

C# é uma linguagem de programação multiparadigma de propósito geral. C# abrange disciplinas de programação estática, forte, com escopo lexical, imperativa, declarativa, funcional, genérica, orientada a objetos e orientada a componentes.


R é uma linguagem de programação para computação estatística e gráficos suportados pelo R. O R é usado entre mineradores de dados e estatísticos para análise de dados e desenvolvimento de software estatístico.
